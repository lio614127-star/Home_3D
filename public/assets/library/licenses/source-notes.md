# 3D Asset Source and License Metadata

This directory contains the starter 3D assets for the Home 3D Editor.

## Sources

### 1. Kenney Furniture Kit
- **Source**: [https://kenney.nl/assets/furniture-kit](https://kenney.nl/assets/furniture-kit)
- **License**: CC0 1.0 Universal (Public Domain)
- **Status**: Currently returning 404. Replaced with generated placeholders for Phase 1.5.

### 2. Generated Placeholders
- **Source**: Local Node.js Script using Three.js procedural geometry
- **License**: Project Internal
- **Attribution Required**: No
- **Categories**: chair, table, sofa, bed, cabinet, plant pot, tree, lamp, door, window.
- **Note**: These are simple low-poly `.glb` files used for testing the asset pipeline before importing heavy real-world assets.

### 3. ambientCG
- **Source**: [https://ambientcg.com/](https://ambientcg.com/)
- **License**: CC0 1.0 Universal (Public Domain)
- **Attribution Required**: No
- **Categories**: PBR Materials (Concrete, Wood Floor, Roof Tiles, Grass, Stone, Brick)

## Manifest

All assets are tracked in `asset-manifest.json` which includes exact source URLs and licenses for every item downloaded.
